# 📊 Dashboard de Inteligência de Mercado e Vendas

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um dashboard em Power BI voltado para análise de vendas e inteligência de mercado.

O objetivo é transformar dados brutos em informações estratégicas, permitindo o acompanhamento de indicadores de desempenho, identificação de oportunidades de negócio e suporte à tomada de decisão.

## 🎯 Objetivos

* Monitorar o desempenho de vendas.
* Analisar faturamento e volume de vendas.
* Identificar tendências de mercado.
* Comparar resultados entre períodos.
* Apoiar decisões estratégicas baseadas em dados.

## 🛠️ Ferramentas Utilizadas

* Power BI
* Excel
* Power Query

## 📈 Indicadores Analisados

* Faturamento Total
* Quantidade de Vendas
* Ticket Médio
* Evolução das Vendas
* Produtos Mais Vendidos
* Participação por Categoria
* Análise Temporal
